# 项目简介 python60

本项目是一个基于Python的图书推荐管理系统，运用了协同过滤算法进行智能推荐。

## 技术栈

本项目采用了以下技术栈：

- **前端**：HTML, CSS, JavaScript, jQuery, Bootstrap, ECharts, Vue
- **数据库**：MySQL
- **后端**：Django, FastAPI, Flask

## 功能模块

系统主要包括以下功能模块：

1. 用户管理
2. 图书信息管理
3. 推荐算法实现
4. 数据可视化

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.x
- 建议使用Linux或macOS操作系统，以获得最佳性能

## 部署步骤

1. 安装Python环境
2. 安装依赖库
3. 配置数据库
4. 部署前端和后端服务
